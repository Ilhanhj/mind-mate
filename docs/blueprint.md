# **App Name**: MindMate

## Core Features:

- Mood Tracking: Log daily moods using emojis, a mood intensity slider, and optional notes.
- Breathing Exercises: Guide users through timed breathing exercises with calming animations and optional background music.
- Guided Journaling: Provide prompts and a text box for users to reflect on their day; allow voice journaling.
- Mood Trend Analysis: Present weekly mood trends and frequent mood tags to help users understand their emotional patterns.
- Personalized Insights: Leverage a generative AI tool to generate personalized insights and recommendations based on mood trends and journal entries.

## Style Guidelines:

- Primary color: Soft light blue (#A9C9FF) for a calming, friendly feel.
- Background color: Very light desaturated blue (#F0F8FF).
- Accent color: Lavender (#FFBBEC) to complement the blue and add a gentle, comforting touch.
- Font: 'Poppins', a geometric sans-serif for a modern and readable interface.
- Use rounded corners (2xl radius) and generous whitespace to create an uncluttered layout.
- Implement gentle micro-animations for breathing exercises and screen transitions to enhance user experience.
- Employ minimal, line-based icons for a clean and friendly interface.
